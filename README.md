# Dotfiles
Storing some configuration.

In your home dir:

    git clone git@github.com:alockwood05/dotfiles.git .vim```

    ln -s .vim/.vimrc .vimrc

    ln -s .vim/.zshrc .zshrc

    git clone git@gtithup.com:powerline/fonts.git .vim/fonts

    ./.vim/fonts/.install.sh

Inside vim:

    :PlugInstall
